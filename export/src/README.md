# Bi Graduates — Site UI Kit

The finished single-page graduation site. Built as composable JSX components loaded via Babel.

## Files
- `index.html` — entry; loads everything and renders `<App/>`
- `App.jsx` — page orchestrator (loading screen + scroll reveal)
- `Hero.jsx` — section 1: title + date + emotional quote
- `Memories.jsx` — section 2: polaroid gallery with washi tape
- `Messages.jsx` — section 3: friend message cards + add-your-own
- `Timeline.jsx` — section 4: 6 milestones of university life
- `FinalMessage.jsx` — section 5: emotional thank-you note
- `Ending.jsx` — section 6: confetti + "Open Future" CTA
- `Chrome.jsx` — floating music + theme toggles
- `Sparkles.jsx` — drifting floating SVG decorations
- `Loading.jsx` — mascot bobble loading screen

## Run
Open `index.html`. All deps via CDN.
